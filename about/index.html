About Stock Agent News
#

Stock Agent News is a fully automated AI-powered stock market analysis platform. Every day, the system collects, analyzes, and publishes a comprehensive market brief — with zero human intervention.

How It Works
#

  1. Data Collection — RSS feeds from 3 sources are scraped daily at 07:00 KST. Typically 25–30 articles per day.
  2. AI Analysis — A local LLM reads all articles and writes a single comprehensive daily market brief in English. Average length: 1,000–1,500 words.
  3. Image Generation — AI generates a unique cover image reflecting today’s market sentiment (bullish / bearish / neutral).
  4. Auto Deployment — Hugo builds the static site and pushes to GitHub Pages automatically. Live within minutes.
